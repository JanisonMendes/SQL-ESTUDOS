# 📊 SQL Estudos: Do Zero ao Avançado

Bem-vindo ao meu repositório de estudos de SQL! Aqui documento minha jornada aprendendo a manipular bancos de dados, desde consultas simples até subconsultas complexas e modelagem.

## 🚀 O que você vai encontrar aqui?

Este repositório está organizado por tópicos para facilitar o aprendizado e a consulta rápida:

* **Fundamentos:** `Selecionar.sql`, `Limite.sql`, `Ordenby.sql`.
* **Filtros e Operadores:** `Operadores.sql`, `Operlogicos.sql`, `Opecomparacao.sql`.
* **Agregações e Agrupamentos:** `FuncaodeAgregacao.sql`, `AgruparPor.sql`, `Tendo.sql (HAVING)`.
* **Relacionamentos:** `Joins.sql` e exercícios específicos de `ExercíciosJoin.sql`.
* **Avançado:** `Subconsulta.sql` e diversas `Funções.sql`.
* **Prática:** Pastas de exercícios para fixação do conteúdo.

## 🛠️ Tecnologias Utilizadas

* **Banco de Dados:** PostgreSQL 🐘
* **Ferramenta de Gerenciamento:** pgAdmin 4 🖥️
